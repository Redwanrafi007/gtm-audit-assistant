GTM Audit Assistant is a Chrome extension designed to help users audit Google Tag Manager containers by analyzing tags, triggers, and variables.

## Data Collection
The extension does not collect personal information, browsing history, website content, or user activity.

## Authentication
A temporary session token is stored locally using `chrome.storage.local` to maintain the user session. This token is not shared, sold, or used for tracking purposes.

## Data Usage
All actions performed by the extension are explicitly initiated by the user. Data is used only to generate audit results for Google Tag Manager containers.

## Third-Party Sharing
The extension does not sell, rent, or transfer user data to third parties.

## Security
The extension follows industry best practices to ensure secure communication with its backend service.
